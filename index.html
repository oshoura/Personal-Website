About me

I'm a third year cs student at Northeastern University studying Computer Science with a concentration in Artificial Intelligence. Some of my favorite courses were/are Object Oriented Design, Programming Languages, Artificial Intelligence, and Machine Learning. Graduation: May 2024

First coop - Machine Learning at Liberty Mutual

My first coop was at Liberty Mutual as a machine learning engineer. Most of my work was in building MLOps tools in python to help speed up model deployments. I redesigned a modeling library to reduce memory usage, improve documentation and standardize interfaces by following sklearn standards. I was able to make the library more user-friendly for the 12+ data scientists who rely on it. I also designed a new library, with two other engineers. I built data transformers and automation tools which sped up deployment of underwriting models from week and months to days. I presented these tools to external teams and received very positive feedback. Overall, I gained an understanding of how machine learning operates in the real world and the struggles engineers face. I gained the skills, tools, and confidence that will undoubtedly help my career down the line while adding measurable value to the Liberty Mutual in my 6 months there

Software Engineering at Northeastern

An organization I got involved with as a software engineer was Generate: Student-led Product Development Studio. There, I collaborated with 5 engineers to build a React and Django-based web app prototype in 4 months. I set up the authentication process on the front end, Spotify user integration, and Redux for state management. I also implemented the user interface for the home screen and explore page in React according to Figma designs. We worked closely with our client, Corbal Distribution who was ultimately very happy with the application when we presented with them. They are continuing development of it.
